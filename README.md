# IntelliUI

AI-native React infrastructure for modern AI applications.

IntelliUI is a React ecosystem focused on building reusable AI interaction systems and conversational frontend primitives for modern AI-powered applications.

Unlike traditional UI libraries designed for generic web interfaces, IntelliUI focuses specifically on:

* streaming AI interactions
* conversational UX
* markdown rendering
* syntax-highlighted code blocks
* reusable orchestration systems
* provider-aware interfaces
* AI-native frontend primitives

---

# Current Package

## intelliui-ai-chat

The first published package in the IntelliUI ecosystem.

Features:

* Real-time streaming
* Markdown rendering
* Syntax-highlighted code blocks
* AIProvider
* useChat hook
* Modular renderer architecture
* Smart auto-scroll
* Headless customization support
* Secure server-side integration support

npm:
https://www.npmjs.com/package/intelliui-ai-chat

# Ecosystem Vision

IntelliUI is evolving toward a collection of AI-native React primitives such as:

* AIChat
* AIInput
* AIMessage
* AICodeBlock
* AIReasoning
* AICitations
* AIFileUpload
* AISearchAssistant

The goal is to make AI application frontend development:

* reusable
* composable
* installable
* developer-friendly

without rebuilding conversational UX systems from scratch.
